## 📖 게임 소개

아이언돔은 실제 미사일 방어 시스템을 모티브로 한 3D 전략 시뮬레이션 게임입니다. 플레이어는 도시를 보호하기 위해 미사일 발사대와 공장을 전략적으로 배치하여 적의 미사일 공격을 막아내야 합니다.

실행주소1 : https://iron-dome-beryl.vercel.app/

실행주소2 : https://dev-canvas-pi.vercel.app/

### 🎯 게임 목표
- 적 미사일로부터 도시를 보호
- 전략적 자원 관리
- 최대한 많은 웨이브 생존

## 🎮 게임 플레이

### 기본 조작법
- **마우스 드래그**: 카메라 회전
- **마우스 휠**: 줌 인/아웃
- **마우스 클릭**: 발사대/공장 배치
- **ESC 키**: 배치 모드 취소

### 게임 시스템

#### 🏗️ 건설 시스템
- **미사일 발사대** ($100): 6발의 미사일을 보유한 방어 시설
- **미사일 공장** ($100): 미사일을 자동 생산하여 발사대에 공급

#### 💰 경제 시스템
**수입원:**
- 초기 자금: $250
- 웨이브 완료: +$100
- 미사일 요격: +$5 (연속 요격 시 보너스)
- 완벽한 방어: +$75
- 도시 보호 보너스: +$15~50
- 높은 요격률: +$25

**지출:**
- 발사대 구매: $100
- 공장 구매: $100
- 미사일 배송비: $3/회

#### 🎯 보너스 시스템
- **연속 요격 보너스**: 3회 연속 (+$10), 5회 이상 (+$15)
- **완벽한 방어**: 웨이브에서 미사일을 하나도 놓치지 않음 (+$75)
- **도시 보호 보너스**: 도시 체력에 따라 차등 지급
- **높은 요격률**: 80% 이상 요격 시 보너스

#### 🔄 웨이브 시스템
웨이브가 진행됨에 따라 난이도가 증가합니다:
- **미사일 속도**: 웨이브마다 미사일 속도 증가 (0.00002씩 증가)
- **미사일 생성 빈도**: 웨이브가 높을수록 더 빈번하게 미사일 생성
- **동시 생성 미사일 수**: 웨이브 1-2(2개), 3-4(3개), 5-6(4개), 7+(5개)
- **미사일 타입**: 난이도에 따라 다양한 미사일 타입(A, B, C) 등장

#### ⚙️ 업그레이드 시스템
각 범주별 10단계 업그레이드를 통해 방어 능력을 향상시킬 수 있습니다:
- **발사대**: 범위, 미사일 용량, 발사 속도 업그레이드
- **공장**: 생산 속도, 저장 용량, 배송 비용 업그레이드
- **도시**: 건물 내구도, 복구 속도, 수입 증가 업그레이드

#### 📊 플레이어 이력 시각화
게임 오버 시 상세한 플레이 이력을 차트로 확인할 수 있습니다:
- **요격 성공률 그래프**: 웨이브별 방어 성공률
- **건물 손실률 그래프**: 도시 피해 현황
- **자원 효율성 그래프**: 자금 관리 효율
- **업적 시스템**: 게임 플레이에 따른 다양한 업적 달성
- **웨이브 상세 정보**: 각 웨이브별 세부 통계
- **이전 게임과 비교**: 게임 성과 비교 분석
